## ① Instagramにも動画を展開する

**概要**
LP用に作ったデモ動画(現状 `demo.mp4` / 45秒・ナレーション+BGM+字幕)を、
Instagram(リール・ストーリーズ)でも回して集客に使う。

**良い点**
- 既存の `demo.mp4` がそのまま資産になる。字幕構成なので音なし前提のIGと相性が良い。

**注意点 / 要件の違い**
- IGは **9:16 の縦型**が基本(LPは横 or 正方形)。同じ動画を使い回すには縦版が別途必要。
- 冒頭2秒で掴む・音なし前提(字幕必須)がIGの鉄則。
- 「基本的に動画を更新できるなら」が条件 → **更新コストを下げる仕組み**が肝。

**次アクション**
- [ ] 既存デモを 9:16 に再構成できるか検証
- [ ] LP用(横/正方形)とIG用(縦)を**同じ生成パイプラインで同時に吐ける**形にする(→②と直結)
- [ ] 現 `demo.mp4` のアスペクト比を確認

## ② 業種ごとに向けた動画を生成する

**概要**
「問い合わせ→AI下書き→LINE通知→承認→自動返信」という汎用デモ1本ではなく、
**業種 × その業種のよくある面倒作業**でテンプレ化し、動画を量産する。

**良い点 / なぜ筋が良いか**
- 刺さるのは「あなたの業種の、あの作業」という具体。汎用1本より訴求が強い。
- 1つの型(テンプレ)から量産できる → 制作コストが下がる。
- ①と掛け合わせると最強:**業種別 × 縦型**でIGの「保存されるリール」として回せる。
- 将来的にLPも業種別分岐(例: `?industry=beauty`)が可能。

**業種 × 面倒作業の例**
| 業種 | よくある面倒作業 |
|------|------------------|
| 美容室 | 予約リマインド・口コミ返信 |
| 工務店 | 見積書づくり |
| 士業 | 書類作成 |
| 飲食 | 口コミ返信・予約対応 |

**次アクション**
- [ ] 動画テンプレ(台本の型)を1つ定義する
- [ ] 差し替えパラメータ(業種名・作業名・画面イメージ)を決める
- [ ] まず1業種で試作 → 横展開

**メモ:** 3案の中で本命。①②を同時に設計するのが効率的。

## ③ 同業のAIコンサル向け営業メール(下請け提案)

**概要**
エンド顧客向けではなく、**同業のAIコンサル**に対して
「提案・折衝はそちらで、面倒な構築・運用だけこちらで巻き取ります」という
BtoBtoB(下請け/協業)の営業メールを送る。

**良い点**
- アリ。既存の実装力を「手を動かす部分だけ請ける」形で売れる。

**注意点**
- **LPは分けたほうがいい。** 現LP(AIBOU)は「AIが苦手な経営者」向けのトーン。
  同業コンサル向けは訴求が真逆(=実装を巻き取る)。同じLPに混ぜると両方ぼやける。

**訴求の骨子**
- 提案・折衝 → 相手(コンサル)
- 面倒な構築・運用だけ → こちら
- 「案件の"手を動かす部分"だけ請けます」

**次アクション**
- [ ] メール文面のドラフト作成
- [ ] 必要なら同業向けの別LP / 別ページを検討
